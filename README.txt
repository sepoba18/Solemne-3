SpaceX Dashboard - Solemne 3

Instalación y Ejecuión: 

    Para reproducir el funcionamiento del aplicativo, tener todos los archivos en una misma carpeta:
    1. Instalar dependencias: Abrir la terminal en la carpeta del proyecto y ejecutar el comando:
        
        pip install streamlit pandas requests altair

    2. Ejecutar la aplicación:

        python -m streamlit run Solemen3.py
